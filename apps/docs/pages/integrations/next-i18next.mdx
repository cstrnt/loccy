# next-i18next

Using **Loccy** with [next-18next](https://www.npmjs.com/package/next-i18next) is pretty simple

**IMPORTANT**: Please make sure you read the [Getting Started Guide](/getting-started) first!

### Setup

To get started simply install the the `@loccy/next-18next` package using your favourite package manager:

```bash
npm i @loccy/next-i18next
pnpm i @loccy/next-i18next
yarn add @loccy/next-i18next
```

now you only need to update your `next-i18next.config.js` like this:

```javascript
const loccyConfig = require("./loccy.config").default;
const LoccyBackend = require("@loccy/next-i18next").default;

module.exports = {
  use: [LoccyBackend],
  ...LoccyBackend.getRequiredOptions(loccyConfig),
};
```

This is the most basic setup needed. If you want to have nice types as well you will have to create
a `next-i18next.d.ts` file in your project root and add the following content:

```typescript
import "next-i18next";
import loccyConfig from "./loccy.config";
import { inferLocales } from "@loccy/next-i18next";

declare module "next-i18next" {
  interface Resources extends inferLocales<typeof loccyConfig> {}
}
```

Now your app is fully typed!
