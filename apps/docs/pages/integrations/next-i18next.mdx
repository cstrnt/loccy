# next-i18next

Using **Loccy** with [next-18next](https://www.npmjs.com/package/next-i18next) is pretty simple

**IMPORTANT**: Please make sure you read the [Getting Started Guide](/getting-started) first!

### Setup

To get started simply install the the `@loccy/next-18next` package using your favourite package manager:

```bash
npm i @loccy/next-i18next
pnpm i @loccy/next-i18next
yarn add @loccy/next-i18next
```
